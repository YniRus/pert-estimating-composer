# Инициализация кастомного DNS API для Timeweb Cloud

Этот каталог содержит скрипт инициализации для кастомного провайдера DNS API Timeweb Cloud (`dns_timeweb.sh`).

Поскольку данный скрипт является внешней зависимостью, он не хранится в основном репозитории проекта и должен быть загружен перед использованием.

## Инструкция по установке

Для того чтобы скачать скрипт и подготовить его к работе, запустите скрипт `init.sh` из этого каталога:

```bash
./dnsapi/dns_timeweb/init.sh
```

После выполнения этой команды файл `dns_timeweb.sh` будет загружен в родительскую директорию `dnsapi/` и при следующем запуске `acme-init.sh` автоматически пробросится в контейнер `acme.sh`.
